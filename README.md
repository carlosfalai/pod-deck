# Pod Deck

Virtual RODECaster soundboard for Windows. 32 sound pads, 5 channel strips, multitrack recording.

![Screenshot](screenshot.png)

## What is Pod Deck?

Pod Deck is a free virtual soundboard and podcast mixer for Windows that replicates the RODECaster Pro experience — without the hardware. Load sound effects, control channel strips, and record multitrack audio sessions directly from your desktop.

## Features

- 32 triggerable sound pads with custom labels and colors
- 5 channel strips with faders and mute/solo controls
- Multitrack recording to separate audio files
- Import/export pad banks
- Always-on-top overlay mode for streamers and podcasters
